# wordle-guesser

Picks a random english word with a given length from a dictionary and then tries to guess the word in as few tries as possible (no limitation).

## Example output

Target word: 'brink'
Guess 1:     'oreas'
Guess 2:     'tripl'
Guess 3:     'grind'
Guess 4:     'briny'
Guess 5:     'brink'
